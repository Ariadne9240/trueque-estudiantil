# Plataforma de Trueque Estudiantil 📚🤝

Una plataforma digital donde estudiantes pueden intercambiar servicios como tutorías, asesorías, apuntes o materiales de estudio, sin necesidad de dinero.

---

## 🚀 Objetivo

Fomentar el aprendizaje colaborativo y el apoyo mutuo entre estudiantes a través del trueque de conocimientos y recursos.

---

## 💡 Ejemplos de Trueques

- Tutoría de matemáticas por asesoría en inglés.
- Apuntes de biología por diseño de portada de tesis.
- Clases de programación por prácticas de oratoria.

---

## 🛠️ Herramientas Utilizadas

- 🗨️ Foro (Flarum o Discourse) para publicar y solicitar trueques.
- 🗂️ GitHub para organizar recursos y documentación.
- 📋 Formularios de Google para registrar a los participantes.

---

## 🌐 ¿Cómo participar?

1. Ingresa al foro (enlace pendiente).
2. Regístrate y publica qué ofreces y qué necesitas.
3. Contacta con otros estudiantes para acordar el intercambio.
4. Registra el acuerdo en el formulario.
5. (Opcional) Documenta el resultado en este repositorio.
